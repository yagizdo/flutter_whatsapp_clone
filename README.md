# Flutter - Whatsapp Clone

Whatsapp Clone App I made with Flutter.


## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/flutter_whatsapp_clone
```

Go to the project directory

```bash
  cd flutter-whatsapp-clone-master
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
